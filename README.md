# 极简天气 🌤️

一款简洁优雅的纯前端天气查询应用，支持PC端和移动端访问。

## ✨ 功能特性

- **实时天气查询**：获取当前天气状况，包括温度、湿度、风速等
- **24小时预报**：展示未来24小时逐小时天气变化
- **未来几天预报**：提供5天天气预报
- **详细气象数据**：显示气压、能见度、空气质量、降水概率等
- **城市搜索**：支持中文城市名搜索，智能提示
- **城市收藏**：收藏常用城市，快速切换
- **默认城市设置**：设置默认显示城市
- **社交分享**：支持分享到微博、QQ，或生成天气卡片图片
- **主题切换**：支持日间/夜间模式
- **响应式设计**：完美适配PC端和移动端

## 🚀 技术栈

- **HTML5**：语义化标签，SEO友好
- **CSS3**：Flexbox布局、动画效果、响应式设计
- **JavaScript ES6+**：模块化编程、异步处理
- **OpenWeatherMap API**：天气数据来源
- **LocalStorage**：本地数据持久化

## 🛠️ 快速开始

### 本地运行

1. 克隆或下载项目到本地
2. 使用浏览器直接打开 `index.html` 文件即可运行

### API配置

项目使用OpenWeatherMap API获取天气数据，API密钥已内置在 `script.js` 中：

```javascript
const Config = {
    API_KEY: '3fd512542df3df1514995b577b973328',
    API_BASE_URL: 'https://api.openweathermap.org/data/2.5',
    DEFAULT_CITY: '北京'
};
```

如需更换API密钥，请修改 `Config.API_KEY`。

## 🎯 使用说明

### 搜索城市

1. 在搜索框中输入中文城市名（如：北京、上海、广州）
2. 点击搜索按钮或按回车键进行查询
3. 支持搜索建议自动补全

### 收藏城市

1. 在天气详情页点击城市名称旁边的心形按钮
2. 收藏的城市会显示在收藏列表中
3. 支持设置默认城市

### 分享天气

1. 点击分享按钮打开分享菜单
2. 选择分享平台（微博、QQ）或复制链接
3. 支持生成天气卡片图片下载

### 主题切换

点击右上角的月亮图标切换日间/夜间模式。

## 🎨 天气效果

应用根据天气状况显示不同的背景效果：
- ☀️ 晴天：明亮渐变背景
- ☁️ 多云：柔和灰色调
- 🌧️ 雨天：蓝色渐变背景
- ❄️ 雪天：白色冷色调
- ⛈️ 雷暴：深色背景
- 🌫️ 雾天：朦胧效果
